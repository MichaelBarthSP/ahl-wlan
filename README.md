# AHL WLAN Setup for Windows
## 1. Open the `Control Panel` (`Systemsteuerung` in German)
![Open Control Panel](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/010-open_controlpanel.png?raw=true)

## 2. Open `Network and Internet`
![Open Network and Internet](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/020-network_settings.PNG?raw=true)

## 3. Open `Network and Sharing Center`
![Open Network and Sharing Center](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/030-network_sharing_center.PNG?raw=true)

## 4. Click on `Set up a new connection or network`
![Setup a new connection or network](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/040-setup_connection.PNG?raw=true)

## 5. Select `Manually connect to a wireless network` and click `Next`
![Manually connect to a wireless network](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/050-manually_connect.PNG?raw=true)

## 6. Enter shown information below
![Basic wlan options](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/060-wireless_information.PNG?raw=true)

## 7. Click on `Change connection settings`
![Change connection settings](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/070-change_settings.PNG?raw=true)

## 8. Open `Settings`
![Settings](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/080-settings_button.PNG?raw=true)

## 9. Uncheck `Verify the server's identity by validating the certificate` and click `Configure...`
![Do not verify the certificate](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/090-configure.PNG?raw=true)

## 10. Uncheck (if checked) `Automatically use my Windows logon name and password (and domain if any)` and click `OK`
![Do not try Windows logon credentials](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/100-no_windows_logon.PNG?raw=true)

## 11. Click `OK`
![Click OK](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/110-ok.PNG?raw=true)

## 12. Open `Advanced Settings`
![Open Advanced Settings](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/120-advanced_settings.PNG?raw=true)

## 13. Check `Specify authentication mode:` and select `User authentication` and click the `Save credentials` button
![Specify authentication mode](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/130-auth_mode.PNG?raw=true)

## 14. Enter the credentials you got after filling out the AHL network registration form
![Enter credentials](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/140-credentials.PNG?raw=true)

## 15. Click `OK` to close the window
![Click OK](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/150-ok.PNG?raw=true)

## 16. Click `OK` to close the window
![Click OK](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/160-ok.PNG?raw=true)

## 17. Click `Close` to close the window
![Click Close](https://github.com/MichaelBarthSP/ahl-wlan/blob/master/170-close.PNG?raw=true)

## 18. Normally windows should now automatically connect to the `AHL` wlan network
